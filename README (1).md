git push -u origin main# MPUZA - Professional Connection Login

A modern login form interface for the MPUZA professional connection platform.

## Features

- Clean and professional login form design
- Email and password input fields with icons
- Remember Me checkbox
- Password visibility toggle
- Responsive design with custom styling
- Professional header and footer sections

## File Structure

```
frontend/
├── index.html      # Main HTML file with form structure
├── style.css       # CSS styling for the login page
├── elephant.png    # MPUZA logo image
└── README.md       # This file
```

## Components

### Header
- Red/orange colored header with "Mpuza" branding

### Form
- Logo container with elephant image
- Title: "Connect us professionally"
- Email input with envelope icon
- Password input with key icon and eye toggle icon
- Remember Me checkbox
- Sign In button (pill-shaped)

### Footer
- Blue footer displaying "Kigali, Rwanda"

## Technologies Used

- HTML5
- CSS3
- Font Awesome Icons (v6.4.0)

## Styling Highlights

- Soft shadows and rounded corners for modern appearance
- Blue accent color (#3498db) for interactive elements
- Responsive form container with proper spacing
- Icon positioning with absolute positioning
- Hover effects on input fields and buttons

## Installation

No installation required. Simply open `index.html` in a web browser.

## Usage

1. Enter your email address
2. Enter your password
3. Optionally check "Remember Me"
4. Click "Sign In" to submit the form
